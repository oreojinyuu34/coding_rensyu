デジタルハリウッドSTUDIO米子で学べるWebコーディングの言語

HTML

Hyper Text Markup Languageの略。 Webサイトを表示するアプリケーション「ブラウザ」において文章構造を正しく読み取るために開発されたマークアップ言語。 近年では画像のレスポンシブに対応したり、インタラクションのあるタグが登場したりと進化が続いている。

非表示にする

CSS

Cascading Style Sheetsの略。 HTMLで構成したドキュメントの装飾を行うための言語。 CSS3から実装できることが爆発的に増え、レイアウトに特化したものや、アニメーションのためのプロパティなどが多数追加された。

JavaScript

Netscape Navigator 2.0で動作することを目指して作成されたプログラミング言語。 古くはアニメーション演出などで使われたが一時期はそのセキュリティのリスクから敬遠されていた過去がある。 紆余曲折を経て現在はブラウザのみならずサーバーサイドなど幅広い箇所で活用されている。
